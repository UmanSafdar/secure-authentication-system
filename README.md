# Secure Authentication System

A secure PHP-based authentication system using sessions and password hashing.

## Features
- User Registration
- User Login
- Session-Based Access Control
- Protected Welcome Page
- Logout Functionality
- Password Hashing using password_hash()
- Password Verification using password_verify()

## Technologies Used
- PHP
- MySQL
- HTML
- Bootstrap
- XAMPP

## How to Run the Project

1. Install XAMPP.
2. Start Apache and MySQL from XAMPP Control Panel.
3. Copy the project folder into:
   C:/xampp/htdocs/
4. Open phpMyAdmin:
   http://localhost/phpmyadmin
5. Create a new database.
6. Import the `database.sql` file into the database.
7. Update database credentials inside `db.php`.
8. Open your browser and visit:
   http://localhost/your-folder-name/login.php

## Database Setup

- The `database.sql` file contains the required table structure.
- Import it before running the project.
